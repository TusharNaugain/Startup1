HOW TO START THE APP
======================

*** FIRST TIME SETUP (If you just unzipped this) ***
1. Open Terminal.
2. Go to this folder:
   cd /path/to/Liberty
3. Create a python virtual environment (do this once):
   python3 -m venv venv
4. Install dependencies:
   source venv/bin/activate
   pip install -r requirements.txt
   
*** HOW TO RUN (Daily) ***

1. Open your Terminal.

2. Go to the Liberty folder:
   cd /Users/tushar/Downloads/hmtl/Liberty

3. Turn on the engine (activate environment):
   source venv/bin/activate

4. Start the App:
   python app.py

5. Open this link in your browser:
   http://127.0.0.1:5000

6. use this to kill the terminal:
   lsof -ti :5000,5001,5002 | xargs kill -9

That's it!
